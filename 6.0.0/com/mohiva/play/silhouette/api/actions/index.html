    A secured request handler.

    A secured request handler.

    A handler which intercepts requests and checks if there is an authenticated user. If there is one, the execution continues and the enclosed code is invoked.

    If the user is not authenticated or not authorized, the request is forwarded to the com.mohiva.play.silhouette.api.actions.SecuredErrorHandler.onNotAuthenticated or the com.mohiva.play.silhouette.api.actions.SecuredErrorHandler.onNotAuthorized methods.
